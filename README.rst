Install
------------------------------------------------------------------------------

``learn_airflow`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install learn_airflow

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn_airflow
